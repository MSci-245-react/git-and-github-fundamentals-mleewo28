### In lecture 1 I learned that the comparison operator `==` in python does not serve the same prupose in Javascript. 

Key differences between `==` and `===`in Javascript: 
- `==` compares values only, converting types if needed (type coercion) 
- `===` compares both value and type (no conversion happens) 
- It is prefered to use `===` and `!==` in Javascript to avoid unexpected behavior

```javascript
"732"==732
```
